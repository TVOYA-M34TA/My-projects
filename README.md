## Обо мне

Создаю Telegram-боты и веб-интерфейсы, которые реально упрощают жизнь людям.

Что мне важно:
- Чистый, читаемый и производительный код
- Удобство для пользователей
- Минималистичный, но современный дизайн
- Когда бот экономит время, нервы и деньги
